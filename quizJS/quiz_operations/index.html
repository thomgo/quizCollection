Les opérations de base

Qu'est ce que la concaténation ?

La concaténation c'est l'action de prendre plusieurs chaînes de caractères pour n'en former plus qu'une. Elle permet notamment de récupérer les données aux fur et à mesure puis de les regrouper dans une seule et unique variable de type string

Je souhaite concaténer deux variables (firstName et lastName) dans une nouvelle variable fullName, que dois-je écrire ?

L'opérateur qui permet d'effectuer une concaténation est le signe +, la bonne réponse était donc la deuxième. Si vous y réfléchissez concaténer revient à additionner des chaînes de caractères.

Que va m'afficher l'instruction suivante: alert(42/7); ?

Pour peu que les données soient du type number, il est tout à fait possible de réaliser des opérations mathématiques en JavaScript. Ici nous afficherons donc le résultat de l'opération, soit 6.

Je souhaite stocker dans une variable result le résultat de 25*115, que dois-je écrire ?

Pour réaliser une opération mathématique, les données doivent être de type number et non string. Il suffit alors de déclarer une nouvelle variable avec pour contenu l'opération à réaliser, la bonne réponse était donc la dernière.

Que sont les méthodes et les propriétés ?

Les méthodes et les propriétés sont un ensemble d'opérations pré-enregistrées. Les propriétés sont liées à un certain type de données, la longueur est par exemple une propriété des données de type string. Les méthodes sont des opérations connues de JavaScript qui permettent de réaliser les transformations les plus courantes sur nos données, par exemple diviser une string en sous-strings.

J'ai une variable clientName de type string et je veux en afficher la longueur à l'écran. Je peux écrire...

La longueur est une propriété des données de type string (entre autres). On l'obtient en ajoutant .length sans parenthèses à la variable concernée, la bonne réponse était donc la dernière.

J'ai une variable name de type string et je veux l'afficher en minuscules. Je peux écrire...

Pour afficher ma string en minuscules, je dois effectuer une transformation, je dois donc utiliser une méthode JavaScript. Cette méthode est .toLowerCase(), elle vient s'ajouter à la variable concernée, le point faisant office de liaison. La bonne réponse était donc la deuxième.
