Les types primitifs

Parmi ces propositions, laquelle n'est pas un type primitif ?

Le binaire n'est pas un type primitif du JavaScript, c'est par contre le langage de base tous les ordinateurs

Combien de de types primitifs comptons-nous en JavaScript ?

En JavaScript, il y a 5 types primitifs: string, number, boolean, null et undefined

A quoi reconnaît-on une donnée de type string ?

On reconnaît du texte en JavaScript à la présence de guillemets "mon texte" ou 'mon texte', il ne faut surtout pas les oublier au risque de faire bugger notre programme

Parmi ces valeurs, laquelle est celle d'un boolean

Un boolean ne peut prendre que deux valeurs : true ou false, la bonne réponse était donc la 2. Null et undefined sont eux-mêmes des types primitifs.

Laquelle des propositions suivantes ne sera pas considérée comme un nombre en JavaScript ?

Un nombre en JavaScript est toujours écrit au format numérique, jamais au format texte.

Que vaut une valeur de type null ?

Null existe et vaut quelque chose, il ne vaut rien ! En JavaScript il est important de savoir si quelque chose n'a pas de valeur. On sait donc ce qu'il y a dedans, il n'y a rien !

Dans quel cas une donnée peut-elle être de type undefined ?

Quand une donnée est de type undefined cela veut dire qu'elle n'a pas été déclarée auparavant, l'ordinateur ne sait donc pas qu'elle existe et ne sait pas de quoi vous parlez.
