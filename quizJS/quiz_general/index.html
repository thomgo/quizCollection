La programmation et JavaScript

Pourquoi parle-t-on de langage de programmation pour JavaScript ?

On dit que JavaScript est un langage de programmation car il permet de produire un programme, autrement-dit une suite d'actions que l'ordinateur réalise selon une situation donnée.

Un programme c'est...

Chaque programme est décomposable en de nombreuses petites actions (instructions) que réalise l'ordinateur. Ces instructions sont souvent réutilisables dans d'autres programmes, c'est ce qui fait leur force !

En programmation, nous produisons...

Nous produisons des algorithmes: des suites d'instructions dans un ordre logique.

Quelle est la différence entre le côté client et le côté serveur ?

La bonne réponse était la 3. Côté client on a tout ce que l'internaute voit sur son navigateur, côté serveur on a tous les calculs qui permettent de générer la page vue par l'internaute.

Qu'est ce qu'on ne peut pas faire avec JavaScript

JavaScript n'a absolument aucun rapport avec le Java, pour développer des programmes dans ce langage il faut l'apprendre.
