Questionnaire d'introduction au CSS

Que signifie CSS ?

CSS signifie Cascading Style Sheet, c'est à dire en français feuilles de style en cascade.

A quoi sert le CSS ?

Il faut rendre à César ce qui est à César, le CSS ne nous permet que de gérer le style de nos éléments HTML, ce qui est déjà pas mal. Il ne nous permet pas d'organiser notre page ou de gérer des données, c'est le javascript qui s'occupe de ça !

A quoi peut-on comparer le CSS

Si le CSS gère nos styles, il gère tout ce qui est visuel il fut donc le considérer comme les vêtements ou la peau de notre page. HTML constitue son squelette et Javascript ses muscles qui l'animent

Laquelle de ces méthodes ne permet pas d'intégrer du CSS dans une page ?

Le CSS ne s'écrit jamais dans un fichier texte classique. Pour vos développements il faut utiliser un éditeur spécialisé et respecter les extensions

Parmi ces méthodes laquelle est la meilleure pour intégrer notre CSS dans nos pages ?

La bonne pratique consiste à écrire tout le CSS dans un fichier style.css indépendant et de le lier à nos pages. Cela permet d'appliquer facilement nos règles sur toutes les pages et de conserver un code clair facilement maintenable

Je dispose d'un fichier style.css que je souhaite intégrer dans une page HTML, que dois-je ajouter dans le head de ma page ?

La bonne réponse est <link rel='stylesheet' href='style.css'>. Pour lier notre feuille de style on utilise la balise link (lien en anglais), on spécifie qu'il s'agit d'une feuille de style et on indique son chemin dans le href

Parmi les éléments suivants, lequel n'est pas géré par le CSS

CSS ne s'occupe pas des attributs, ils sont codés "en dur" dans le HTML ou gérés dynamiquement en Javascript

Qu'est que le MDN

MDN signifie Mozilla Developper Network, il s'agit d'une création de la fondation Mozilla tout comme le célèbre navigateur web

A quoi sert le MDN

Le MDN est une ressource formidable pour tous les métiers du web. On y trouve une documentation de très qualité, entièrement gratuites sur les technologies utilisées actuellement. Elle sera incontournable pendant votre apprentissage
