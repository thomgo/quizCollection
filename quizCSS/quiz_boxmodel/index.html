Le box-model en CSS

Qu'implique la notion de box-model en CSS ?

En CSS le box-model implique que tous les éléments HTML sur la page sont des boîtes qui s'imbriquent les unes dans les autres. Vous devez pensez vos pages comme de grandes poupées russes

Parmi les propriété suivantes laquelle correspond à la hauteur d'un élément ?

La hauteur d'un élément correspond à la propriété height, sa largeur à width. High n'est pas une propriété CSS.

Qu'est ce que le margin d'un élément ?

La margin est une propriété physique de nos boîtes. Elle correspond à la marge extérieure d'un élément, c'est à dire la distance qui le sépare de son voisin.

Quand cela est possible, quelle unité doit-on utiliser pour les tailles de nos éléments ?

Vous pouvez utiliser toutes les unités citées mais quand c'est possible, essayez d'utiliser les pourcentages afin que vos éléments puissent s'adapter à la taille de votre écran.
