Questionnaire sur les bases CSS

Qu'est ce qu'un sélecteur ?

Le sélecteur en CSS permet d'identifier l'élément HTML sur lequel nos règles vont s'appliquer

Qu'est ce qu'une propriété ?

Une propriété CSS permet de modifier un aspect particulier d'un élément HTML comme sa couleur ou sa taille par exemple.

Parmi les propositions suivantes, laquelle n'est pas une valeur reconnue par CSS

CSS connaît énormement d'unités différentes (même les centimètes, les pouces etc) mais les mètres carrés n'en font pas partie

Parmi les proposition suivantes, laquelle permet de sélectionner un titre de niveau 2 ?

Un titre de niveau deux correspond à un élément h2 dans notre document HTML. Pour le sélectionner en CSS il faut lui retirer ses <>

Quel couple propriété/valeur est déclaré correctement ?

La bonne réponse était la première. Il faut toujours déclarer la propriété suivie de : puis de sa valeur et terminer par un point virgule

Que permet de faire le propriété text-align ?

Text-align nous permet de gérer l'alignement du texte dans son container selon que l'on souhaite le placer à gauche ("left"), au centre ("center") ou à droite ("right")

Quelle valeur n'est pas assignable à la propriété font-weight ?

Cette propriété gère l'épaisseur du texte, que celui-ci soit en italique ne la concerna absolument pas

Que permet de gérer la propriété font-size ?

Font fait référence à la police (notre type d'écriture) et size à la taille, la propriété font-size permet donc de gérer la taille de nos polices
